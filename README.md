# Bellabeat — Product Analytics Case Study (R)

Portfolio-style product analytics case study using a public wearable-activity dataset. Exploratory analysis in R, translated into engagement hypotheses and measurement-ready recommendations.

## Quick links
- **Live page (GitHub Pages):** https://valescabravo.github.io/Bellabeat/
- **Full report (HTML):** https://valescabravo.github.io/Bellabeat/Bellabeat_Upgraded_Portfolio.html
- **Repository:** https://github.com/ValescaBravo/Bellabeat

## Business context
Bellabeat is positioned as a wellness brand. This educational case study explores wearable-activity patterns and frames insights as hypotheses and proposed actions.

> Note: This is **not** Bellabeat internal data. It uses a public dataset for learning purposes.

## Objectives
- Understand behavioural usage patterns in wearable activity data
- Identify potential engagement and retention opportunities
- Translate findings into stakeholder-ready recommendations with measurement ideas

## What this demonstrates
- Data preparation and basic quality checks
- Exploratory analysis (EDA) in **R / RStudio**
- Clear insight-writing and recommendation framing
- Product thinking: hypotheses → actions → metrics
